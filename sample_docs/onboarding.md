# Onboarding Guide

## Welcome to the team!

### Day 1 Checklist
- Set up laptop with IT (contact helpdesk@company.com)
- Get access to GitHub, Jira, Confluence
- Join Slack channels: #general, #engineering, #announcements

### Tools we use
- **Code**: VS Code or IntelliJ
- **Version Control**: GitHub (github.com/company)
- **Project Management**: Jira
- **Documentation**: Confluence
- **Communication**: Slack

### Engineering Practices
- All PRs require 2 approvals
- Write tests for all new features
- Follow conventional commits format
- Daily standups at 9:30 AM EST
